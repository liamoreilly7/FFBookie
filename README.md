## Fantasy Football Bookie

This package creates a sportsbook for ESPN Fantasy Football Leagues. Currently only supported for the Esteemed Domination League.

This package relies heavily on [cwendt94/espn-api](https://github.com/cwendt94/espn-api) for their implementations to access ESPN's Fantasy API.

## Installing
With Git:
```
git clone https://github.com/liamoreilly7/espn-api
cd espn-api
python3 setup.py install
```
